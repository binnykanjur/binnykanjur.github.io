Manage your media

October 29, 2019 by Binny Kanjur

In this tutorial, we will use the Media Library screen to manage (view, add & delete) the images of the blog.

Download sample images

The images used in the sample blog are available on GitHub. Download and extract it locally.

  1. Download the sample images zip file.
  2. Unzip the Lorem-Ipsum-Files-master.zip file.

Add Image(s)

  1. Navigate to the Media Library screen
  2. From the toolbar, Select Add Media.
  3. Navigate to the folder where you had extracted the images earlier & select the image(s).
  4. Click Open.

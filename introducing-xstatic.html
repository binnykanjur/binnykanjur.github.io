Introducing xStatic

October 27, 2019 by Binny Kanjur

xStatic (read Ecstatic) is a simple static site generator with which you can generate a static web site, personal profile, or a blog.

Features

The primary focus while developing xStatic was the ease of use, including a simple install & publishing experience. Here's what xStatic provides

  • One-Click installation and automatic updates.
  • One-Click Publish to GitHub pages.
  • Simple user interface.
  • Theming.
  • Simpler content management & backup with SQLite Database.
  • Compose formatted posts/pages with Markdown.
  • Easy export to HTML.

Technology

xStatic is written in C# & uses WPF for UI, ASP.NET Core for theming.

Download

xStatic is currently only available for Windows 10(Version 1803 or above).
